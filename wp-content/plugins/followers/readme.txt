== Description ==

Simple plugin showing your fans/followers counter from various social services



== Usage ==

You need to first put all required API keys and names for the social services you want to use in wp-admin -> Settings -> Followers. After that, add Followers Widget to one of your sidebar areas, and select which elements you want to show.

== Installation ==

Installing "Followers Plugin" can be done either by searching for "Starter Plugin" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the settings screen and configure, as desired.
